抢课管理端

管理课程数据、导入课程库、爬取学校课程

管理员登录

管理口令保存在服务端配置中（默认与 .env 的 COOKIE_KEY 一致）。登录成功后，该账号将用于爬取学校课程信息。

课程库管理

从本地文件导入

导入课程目录

导入开课信息一览表（如通识选修课目录），用于在用户端显示教师、地点、时间等筛选信息。

在线爬取课程

使用已登录的管理员账号，通过 Selenium 自动登录并爬取全校课程数据。爬取完成后会自动刷新课程库。

手动添加课程

如果课程库不完整，可以手动添加。在教务系统选课页面按 F12 → Network → 找到选课请求 → 右键 Copy as cURL，粘贴到下方；或直接粘贴 form data。
